Frontend (React + JavaScript)

src/: Contém o código-fonte principal.
src/components/: Arquivos dos componentes React.
src/hooks/: Funções personalizadas do React.
src/services/: Arquivos de serviços para comunicação com o backend.
src/utils/: Funções utilitárias.

src/visualizacaoDeEventos/: Componentes de visualização de eventos 
    CalendarioEventos.js:  principal para exibir o calendário de eventos.
    ListaEventos.js: Componente para listar os próximos eventos.
    DetalhesEvento.js: Componente para exibir detalhes de um evento específico.
    FiltroEventos.js: Componente para filtrar eventos por tipo (online/presencial).
    index.js: Arquivo de exportação para os componentes acima.

src/app.js: Arquivo principal da aplicação.
public/: Ativos estáticos como index.html, favicon, etc.
package.json: Dependências e scripts do projeto frontend.
README.md: Documentação para a parte frontend.





Backend (Python)


src/: Contém o código principal do backend.
routes/: Definições de rotas.
controllers/: Lógica dos controladores.
models/: Modelos do banco de dados.
config.py: Configurações do sistema.
app.py: Arquivo principal da aplicação Flask.
requirements.txt: Lista de dependências do Python.
README.md: Documentação para a parte backend.
